Drive-Through Application API Documentation

This documentation provides an overview of the APIs available in the Drive-Through Application built with FastAPI and Svelte.

Available APIs

  • GET /

    Returns the home page of the application.

  • POST /orders/

    Create a new order. Requires a JSON body with order_text containing the order details.

  • DELETE /orders/cancel/

    Cancel an existing order. Requires a JSON body with order_no and cancel_text.

  • GET /orders/canceled/

    Retrieve a list of all canceled orders.

  • GET /orders/all/

    Retrieve a list of all active and canceled orders.

  • POST /process_input/

    Process input through OpenAI to determine the action (create or cancel an order). Requires a JSON body with order_text.
